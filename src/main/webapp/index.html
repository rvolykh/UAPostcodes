Сервіс

UA Postal codes - це RESTfull сервіс, за допомогою якого можна отримати інформацію по поштових індексах України, а також інформацію по областях, районах та населених пунктах. Передача інформації здійснюється у JSON форматі. Дані взято із ресурсу: http://ukrindex.ru

Документація по API

Опис: Ви можете отримати список усіх областей України (включаючи АРК).

API виклик:

UAPostcodes/api/regions

Параметри:

Вихідні:

id - ідентифікатор області (відповідає порядковому номеру області)
name - назва області

API виклик:

UAPostcodes/api/districts?id={regionId}

Параметри:

Вхідні:

regionId - ідентифікатор області, якої Ви бажаєте получити список районів.

Вихідні:

id - ідентифікатор району (відповідає порядковому номеру усіх районів різних областей)
name - назва району

API виклик:

UAPostcodes/api/cities?id={districtId}

Параметри:

Вхідні:

districtId - ідентифікатор району, список населених пунктів якого Ви бажаєте получити.

Вихідні:

id - ідентифікатор населеного пункту (відповідає порядковому номеру усіх міст різних районів, областей)
name - назва населеного пункту
postcode - поштовий індекс населеного пункту
